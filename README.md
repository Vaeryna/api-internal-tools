# Internal Tools API

## Technologies
- Langage: TypeScript
- Framework: Fastify
- Base de données: PostgreSQL
- Port API: [votre_port] (configurable)

## Quick Start

1. `docker-compose --profile postgres up -d` 

2. `npm install`
3. `npm start`
4. API disponible sur http://localhost:3000
5. Documentation: http://localhost:[port]/[chemin_docs]
